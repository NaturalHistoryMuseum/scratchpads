Help for the Lucid3 Player

Welcome to the Lucid3 Player, one member of the Lucid family of products. For more information on other Lucid and related programs, please visit the Lucidcentral website at www.lucidcentral.org

Written in Java, Lucid3 is a cross-platform implementation, and will run on any Java-enabled operating system (such as Windows, Macintosh, Unix, Linux, Sun Solaris etc).

Note that the Java Virtual Machine v1.4.2 or greater must be installed before the Lucid3 programs will run. See the topics Java – what does it mean? and How to check if you have Java installed for more information.
