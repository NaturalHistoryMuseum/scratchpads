Restarting a key

At the end of an identification click the Restart button or use the Restart menu item to reset the key for another identification. All features will be unchecked and the Features Chosen and Entities Discarded windows will be emptied.
 
If the menu option Reset Subsets on Restart is checked, the Player will be restarted using the default subset defined by the key builder. If this option is not checked, the key will restart using the current feature and entity subsets.
