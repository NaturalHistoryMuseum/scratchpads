Click on 'Taxonomy' in the Content menu, find the classification you wish to edit in the list and click on 'Edit classification'. Extend the classification by clicking on the plus icon next to a term. Click on
the term itself to edit this term.

The term name will appear to the right of the tree. Below the name you can select the taxonomic rank of this term and its relationship to the parent term. For all relationships other than 'taxonomic child' the taxonomic rank must be set to 'None'. To add further ranks go to 'Settings', enter the missing rank in the respective box and click on 'Update'. Add the reference for the original description by typing the author, year or part of the title and selecting the correct reference from the generated list. Save.

Synonyms as well as vernacular names can be easily identified in the tree because they are displayed in different colours.

You can move taxa within the tree by simple drag and drop. Using the 'Edit' menu you can add new taxa, rename, cut, paste and delete. Using the 'View' menu you can filter the tree for certain taxa.

Tip: If you don't have the option to edit your classification you need to first click on 'Edit vocabulary'. Under 'Settings' check 'Classification vocabulary' and save. Now you should be able to edit this classification.
