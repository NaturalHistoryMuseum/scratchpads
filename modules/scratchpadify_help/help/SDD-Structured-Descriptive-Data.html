Structured Descriptive Data (SDD) can be used to generate interactive keys or taxon descriptions. To generate an interactive key from a character project, the data have to be exported to SDD and the SDD file needs to be imported into key generating software like Lucid.

Only controlled and numeric characters can be exported into SDD, not text characters.

Warning:

  • Importing the SDD file into Lucid software only works with Lucid 3.4 and 3.5, not 3.3.
  • Numeric characters don't work properly at the moment (7. Sept. 2010).
  • Your taxa all need to have the same rank in the classification. Otherwise the import doesn't work.
  • For controlled characters "?" and "-" are not imported (7. Sept. 2010)
