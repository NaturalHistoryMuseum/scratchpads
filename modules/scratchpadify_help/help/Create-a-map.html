There are several ways to create a map. One way is to do a countries map. Another way is to enter specimen data. If you add the longitude and latitude to the location that is linked to the specimen, then a distribution map will automatically appear on the taxon page. For both maps to be visible on taxon pages you need to have a classification and tag the node (i.e. countries map/specimen) with the correct term.

For experienced maintainers it is also possible to created a custom map view.
