By default the mission statement is shown on the front page of a Scratchpad. The statement disappears when a user is logged in.

To edit the mission statement:

  • Go to ‘Administer’ > 'Advanced' > ‘Site configuration > ‘Site information’.
  • Change the text in the ‘Mission’ textbox.

To edit the layout of the front page so that the mission statement is not shown:

  • Go to 'Administer' > 'Advanced' > 'Site building' > 'Themes'.
  • Click on the 'Configure' tab.
  • Under the 'Toggle display' section unselect 'Mission statement' and save the configuration.
