The 'Taxon description' content type is based on the TDWG Species Profile Model (SPM) and can be used to export Scratchpad taxon descriptions to EOL (Encyclopedia of Life) or to created descriptions for the publication module.

To add a taxon description

  • Add the taxon name. The field will confirm whether or not your name matches one from EOL. If it does not you can still use it, but you need to be aware that if you export this description to EOL, it will not show up on a species page, because there currently is none for this species.
  • Link your name to the Scratchpad classification by entering/selecting the name in the taxonomy field or by clicking on the 'Autotag' button. If you use autotagging, delete unnecessay tags, like for example the genus name if the description if for a species.
  • Enter text into the fields as needed. We recommend to use the 'Morphology' field for the actual morphological description of the taxon and 'Diagnostic Description' for the diagnoses. Save.
