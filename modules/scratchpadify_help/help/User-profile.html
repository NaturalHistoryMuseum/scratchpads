The Profile content type is linked to the User module. Each user has a profile which can be edited under 'My Account' in the Navigation menu. The only way to add a profile node via the Content menu is to set the author of the node as "Anonymous". This is done under the advanced options by having the 'Authored by' field in the 'Authoring information' tab' blank.

The profile content type can be customized same as any other content type.
