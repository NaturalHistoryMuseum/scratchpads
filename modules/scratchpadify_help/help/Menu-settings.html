Create a menu item linking to this node by entering a title and choosing a parent menu item. The menu blocks are denoted by "< >". The default option is <Secondary links>, which is usually (depending on your theme) a second line of menu items in the header, below the <Primary links>. If you want a new menu block in the sidebar, you have to create it first (go to Add menu).

Don't add menu items to the <Navigation> block, because it will mess up this block and the links will only be visible to logged in users. Usually the items in a menu block are sorted alphabetically. If you want to have a different order you can sort the items by giving them different 'Weights'.
