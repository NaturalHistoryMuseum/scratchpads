By default, the name of a Scratchpad is shown in the header, to the right of the logo.

To edit the site name:

  • Go to ‘Administer’ > 'Advanced' > ‘Site configuration > ‘Site information’.
  • Change the text in the ‘Name’ textbox.

To edit the layout of the front page so that the site name is not shown:

  • Go to 'Administer' > 'Advanced' > 'Site building' > 'Themes'.
  • Click on the 'Configure' tab.
  • Under the 'Toggle display' section unselect 'Site name' and save the configuration.
