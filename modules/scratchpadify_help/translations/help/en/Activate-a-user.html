• Go to ‘Administer’ > ‘User management’ > ‘List’.
  • You can activate two ways: 1. Mark the users you want to activate, select ‘Unblock the selected user’ from the drop down menu under ‘Update options’, and click on ‘Update’. 2. Click on ‘edit’ to the right of the user you want to activate. Under ‘Account information’, ‘Status’ mark the ‘Active’ radio button. Don't enter a password because this is generated automatically. Choose a user role under ‘Roles’ and click on ‘Save’. As soon as you have activated the account, the new user will receive an email and will be able to log in.
  • If you received a notification message about a new user, you can click on the link in the email message and then under ‘Account information’, ‘Status’ mark the ‘Active’ radio button. Don't enter a password because this is generated automatically. Choose a user role under ‘Roles’ and click on ‘Save’. If you are not already logged in on your site you will need to log in before you can activate the new user. As soon as you have activated the account, the new user will receive an email and will be able to log in.
