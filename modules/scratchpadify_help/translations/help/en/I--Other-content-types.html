On your own, explore the other content types

  • Countries map
  • Phylogenetic tree (There is no Carnidae species tree, so instead copy the text of "MammalPhylogeny.txt" into the 'Nexus or Newick data' field and tag it to Carnus hemapterus.
  • Taxon description. The taxon description content type is based on TDWG's SPM (Specimen Profile Model) standard. Data in this content type can be harvested by EOL if the Scratchpad maintainer signs an agreement with EOL. Within a Scratchpad, taxon descriptions are used as part of the publication module to publish Scratchpad data in a journal. The publication module is still a prototype, but if you ever intend to use it, it would be wise to make use of the taxon description content type for your descriptions.
