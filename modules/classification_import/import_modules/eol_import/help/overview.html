Importing a classification from the EOL classification service is a quick and easy way of getting started with your site. The classifications provided by this service should not be considered to be the definitive classificaiton for a group, but they should help to get you started in creating the definitive classification on your Scratchpad.

The service currently supports a number of classifications from different data sources, including:

  • Catalogue of Life
  • CU*Star
  • ITIS

Please note, this service is VERY SLOW. Please be patient when doing an import.
