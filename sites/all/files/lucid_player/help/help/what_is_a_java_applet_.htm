What is a Java Applet?

An applet is a program written in the Java programming language that can be embedded in an HTML page, in much the same way as images are included. When you use a Java technology-enabled browser to view a page that contains an applet, the applet's code is transferred to your system and executed by the browser's Java Virtual Machine (JVM). The Lucid3 Player is available as a Java applet. This means users of your key do not need to download and install a Lucid Player, as was the case with previous versions.
