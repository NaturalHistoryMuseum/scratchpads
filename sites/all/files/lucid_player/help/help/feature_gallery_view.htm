Feature Gallery View

The feature gallery view provides an alternative view of multistate features. Instead of a series of states being displayed in the tree below a multistate feature, the feature gallery displays a tabular view of thumbnails representing the states of the feature.

Clicking on a thumbnail in the gallery will select it’s related state, the icons overlaid on the thumbnails can be used to open any html or image files attached to the state.

To enable the gallery view, select the Image Gallery option under the Display submenu located within the Features menu.
