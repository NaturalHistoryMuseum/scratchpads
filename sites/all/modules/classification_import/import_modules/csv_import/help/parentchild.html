This format is based on the submission guidelines for ITIS. We have tweaked the format slightly to suit the Scratchpads and the specific requirements of our users.

The difficult step in producing a file of this type, is ensuring that the parent name for each taxon name is EXACTLY the same as another name listed in the file - this includes the authority information of the name.

The additional fields supported by this format are:

  • rank_name
  • unit_name[1-4]
  • unit_ind[1-4]
  • usage
  • accepted_name
  • Unacceptability_reason
  • taxon_author
  • reference
  • page_number
  • vernacular_name
