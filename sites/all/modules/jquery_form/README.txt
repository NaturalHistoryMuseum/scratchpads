This module allows you to make use of the jQuery Form Plugin (http://www.malsup.com/jquery/form/).

It requires the jQuery Update module (http://drupal.org/project/jquery_update).


INSTALLATION:

1) Place this module directory in your modules folder (this will usually be
"sites/all/modules/").

2) Enable the module.


USAGE:

See http://www.malsup.com/jquery/form/#getting-started.
