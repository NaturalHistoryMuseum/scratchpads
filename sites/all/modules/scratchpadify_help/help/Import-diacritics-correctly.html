You need to get your file into UTF-8 format. Open your file in MS Word and then save as 'Plain text (*.txt)'. A box will open in which you can choose 'Other encoding'. Select 'Unicode (UTF-8)' and check in the Preview whether the text looks o.k.

If your original file was something other than *.txt you might need to change the suffix back to what it was.

Another option is to use the program EditPad Lite, which is freely available online. Go to 'Convert', check 'Encode the original data with another character set...', select 'Unicode, UTF-8, check the Preview and click on 'OK'.
