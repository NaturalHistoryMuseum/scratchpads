• Click on 'Taxonomy' in the 'Content' menu and click on ‘Add vocabulary’.
  • Enter a ‘Vocabulary name’ and ‘Description’.
  • Extend the 'Content types' section to unselect the content types which should not be tagged with this classification. By default all content types are selected. When creating content users will be able to tag content of the selected content types to the new classification. Remember that every time you add a new content type you will have to go back to this page to enable this new content type to be tagged.
  • Extend the 'Settings' section.
    • Select 'Tags', if you want to create a vocabulary which allowes free tagging. This is useful for example for keywords, but not for biological classifications because it is too easy to create misspellings.
    • If you have content which relates to several terms in your classification/taxonomy, select ‘Multiple select’.
    • If you want to add a biological classification that follows ITIS standards go to 'Content type' at the bottom of the page and select 'ITIS name' in the drop down menu. This creates a one to one relationship between the names in the vocabulary and the ITIS name content type.
  • Save.
