All content is saved in nodes. Each page, bibliographic citation, image, forum topic etc. is saved in a separate node. Each node has a unique URL, for example http://myspecies.info/node/1234. An alias which replaces the "node/1234" part of the address with the title of the node is automatically created when a node is published on a site. For example http://myspecies.info/node/1234 might become http://myspecies.info/introduction. Users can customise this alias through the ‘Advanced options’> ‘URL path settings’ options associated with each node.
