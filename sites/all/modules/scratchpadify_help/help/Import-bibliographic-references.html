• Hover with the mouse over 'biblio' in the 'Content' menu and click on the import icon which appears to the right of the menu item.
  • Click on 'Browse' to locate and select the file containing the references into your Scratchpad.
  • Select the file type that describes your bibliographic data. The Scratchpads can import Tagged EndNote, RIS, MARC, EndNote 7 XML, EndNote 8+ XML and BibTex formatted bibliographies. These files should be prepared beforehand using the export features of the bibliographic software (e.g. EndNote) that you are using.
  • Bibliographic references can be automatically tagged with taxon names held within a Scratchpad classification. This works through the title and keywords fields. Autotagging needs to be enabled by going to the advanced administration page and clicking on 'Autotag settings' in the 'Site configuration' section. Under 'Default for "Autotag on save"', select "Biblio" and save the configuration.
  • Click the ‘Import’ button once the settings have been confirmed and wait for the import process to complete (this may take several minutes for very large bibliographies).
  • Check the bibliography to ensure that the import process has completed properly. If the import process failed or is unsatisfactory, users can delete the imported bibliography using the content list accessible from the administration menu (‘Administer’ > 'Advanced' > ‘Content management’ > ‘Content’).

Tips:

  • Autotagging doesn't work if you have set your classification as 'required'.
  • If you have problems with importing RIS data, check out the blog entry which deals with this issue.
